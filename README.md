This is a beginner-friendly example, so you can follow along and learn the basics of descriptive and predictive analytics using Python.

## Quick Start

1. **Clone this repo:**
   ```bash
   git clone https://github.com/Arnitah/Descriptive-and-Predictive-Analytics.git
   ```
2. **Install the main Python packages:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Open the notebook to try out the example:**
   ```bash
   jupyter notebook
   ```
